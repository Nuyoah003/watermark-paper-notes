## 📖 About

This repository collects my personal analysis and notes on papers related to **text watermarking for Large Language Models (LLMs)**. Each paper has its own folder containing a detailed analysis covering research background, methodology, experiments, and personal insights.

## 🏷️ Topics Covered

- **Black-box Watermarking** — No access to model weights or logits
- **Neural-based Watermarking** — End-to-end learning frameworks
- **Inference-time Watermarking** — Token distribution manipulation
- **Robustness** — Resistance to removal and detection attacks
- **Fidelity** — Semantic preservation of watermarked text
