# 📚 Watermark Paper Notes

> Personal reading notes and in-depth analysis for LLM text watermarking papers.

## 📖 About

This repository collects my personal analysis and notes on papers related to **text watermarking for Large Language Models (LLMs)**. Each paper has its own folder containing detailed reading notes covering research background, methodology, experiments, and personal insights.

## 🏷️ Topics Covered

- **Post-hoc Watermarking** — Synonym substitution after text generation (AWT, ContextAwareLS, BlackBox, RobustWatermark, DeepTextMark)
- **Neural-based Watermarking** — End-to-end learning frameworks (AWT, REMARK-LLM)
- **Black-box Watermarking** — No access to model weights or logits (BlackBox, ICW)
- **Inference-time Watermarking** — Token distribution manipulation at generation time
- **Robustness** — Resistance to deletion, substitution, paraphrase attacks
- **Fidelity** — Semantic preservation of watermarked text
